GroupMetadata

class GroupMetadata extends Logging

Group contains the following metadata:

Membership metadata:

  1. Members registered in this group 2. Current protocol assigned to the group (e.g. partition assignment strategy for consumers) 3. Protocol metadata associated with group members

State metadata:

  1. group state 2. generation id 3. leader id
